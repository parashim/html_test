体に優しい自然食を提供する。WCB CAFE。無添加の食材を利用したメニューが特徴です。html＋cssで自分用にプログラミングTipsを整理したサイトをいつか作って、一度忘れてもそこで再度、習得できるようにしておくことが目標。そのためにはどこかにWebサーバの間借りとかも感げないといけなくて難しいよね。今のところ、Tipsにしておきたいのは、GoogleAPIとか、edinet、pythonでいうと、lxmlの使い方とか、dict型、の使い方。あとはpandasも整理したい。classの概念はまだあんまり理解できていないかも、cssselectorも整理しておきたいなぁ。あとはmongodbとsqliteもしっかりインストール手順から参照用のツールの使い方とかも含めて整理しておきたいな。

おいしいブレンドコーヒーとお酒、レモンサワーが飲みたい、CSS頑張って勉強した結果をHPで整理したい、javascriptも頑張って日本地図の動的なHPを作ってみる。
